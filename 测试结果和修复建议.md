## ⚠️ 需要修复的问题

### 问题1: 抓取成功率低（42.9%）

#### 现象分析
1. **红色杯子抓取失败**
   - IK求解成功（误差0.015-0.046m）
   - 夹爪闭合后，物体提升高度为 -0.000m
   - 说明物体没有被夹爪抓住

2. **蓝色方块抓取失败**
   - 预抓取位置IK成功（误差0.018m）
   - 抓取位置IK失败（误差0.068m > 0.05m阈值）
   - 目标位置可能超出工作空间或IK求解困难

#### 根本原因

**1. 物理参数不足**
```xml
<!-- 当前配置 -->
<geom name="cup_body" type="cylinder" size="0.025 0.04" 
       rgba="1.0 0.3 0.3 1" density="500" 
       friction="1.5 0.1 0.1"/>
```
- **摩擦系数过低**: `friction="1.5 0.1 0.1"` - 夹爪和物体之间的摩擦不足
- **物体密度偏低**: `density="500"` - 物体质量小，容易被夹爪推开
- **夹爪接触面积小**: 圆柱体半径0.025m，与夹爪接触面积有限

**2. IK求解精度不足**
```python
# 当前配置
max_iter: int = 100
tol: float = 0.02
damping: float = 0.1
```
- **迭代次数少**: 100次可能不足以收敛到最优解
- **阻尼系数大**: 0.1可能导致求解精度下降
- **容差宽松**: 0.02m（2cm）对于精密抓取过大

**3. 抓取策略问题**
```python
# 当前策略
grasp_pos[2] -= 0.02  # 物体下方2cm
self.step(400)  # 夹爪闭合后等待400步
lift_pos[2] += 0.12  # 直接提升12cm
```
- **抓取深度不足**: 2cm可能无法确保夹爪完全包裹物体
- **等待时间短**: 400步（约0.4秒）可能不足以让物体稳定
- **提升过快**: 直接提升12cm可能导致物体滑落

#### 修复方案

**方案1: 增强物理参数**

### 问题3: IK求解器精度不足

#### 现象
```
[4/7] 执行: 抓取蓝色方块
  阶段3: IK 求解抓取位置 (0.080, -0.080, 0.871)
    [错误] IK 求解失败，误差: 0.068m
```

#### 根本原因
1. **目标位置超出工作空间**: `(0.080, -0.080, 0.871)` 可能接近机械臂可达边界
2. **关节限位限制**: 某些关节可能达到限位
3. **局部最优陷阱**: 阻尼最小二乘法可能陷入局部最优

#### 修复方案

**方案1: 使用更高级的IK算法**
```python
from scipy.optimize import least_squares

class AdvancedIKSolver:
    def solve(self, target_pos: np.ndarray, current_q: np.ndarray = None):
        if current_q is None:
            current_q = np.zeros(6)
        
        def residual(q):
            self.data.qpos[:6] = q
            mujoco.mj_forward(self.model, self.data)
            ee_pos = self.data.site_xpos[self.ee_site_id]
            return target_pos - ee_pos
        
        def jacobian(q):
            jacp = np.zeros((3, self.model.nv))
            mujoco.mj_jacSite(self.model, self.data, jacp, None, self.ee_site_id)
            return jacp[:, :6]
        
        # 使用 Levenberg-Marquardt 算法
        result = least_squares(
            residual,
            current_q,
            jac=jacobian,
            method='lm',
            max_nfev=200,
            ftol=1e-6,
            xtol=1e-6
        )
        
        q = np.clip(result.x, 
                   [-1.7, -0.98, -2.0, -2.0, -2.1, -6.28],
                   [1.7, 1.0, 1.3, 2.0, 2.1, 6.28])
        
        # 验证解
        self.data.qpos[:6] = q
        mujoco.mj_forward(self.model, self.data)
        ee_pos = self.data.site_xpos[self.ee_site_id]
        error = np.linalg.norm(target_pos - ee_pos)
        
        return q, error
```

**方案2: 添加可达性检查**
```python
def is_reachable(self, target_pos: np.ndarray) -> bool:
    """检查目标位置是否在工作空间内"""
    # 简单检查：目标是否在合理范围内
    x, y, z = target_pos
    
    # 根据机械臂参数调整这些值
    max_radius = 0.5  # 最大工作半径
    min_height = 0.6  # 最低高度
    max_height = 1.3  # 最高高度
    
    xy_distance = np.sqrt(x**2 + y**2)
    
    if xy_distance > max_radius:
        return False
    if z < min_height or z > max_height:
        return False
    
    return True

# 在抓取前检查
if not self.is_reachable(target_pos):
    print(f"  [警告] 目标位置 {target_pos} 超出工作空间")
    return GraspResult(...)
```

**方案3: 多起点IK求解**
```python
def solve_multi_start(self, target_pos: np.ndarray) -> Tuple[np.ndarray, float]:
    """从多个起点求解，选择最优解"""
    start_points = [
        np.zeros(6),
        np.array([0.5, 0, 0, 0, 0, 0]),
        np.array([-0.5, 0, 0, 0, 0, 0]),
        np.array([0, 0.5, 0.5, 0, 0, 0]),
    ]
    
    best_q = None
    best_error = float('inf')
    
    for start_q in start_points:
        q, error = self.solve(target_pos, start_q)
        if error < best_error:
            best_error = error
            best_q = q
    
    return best_q, best_error
```

### 问题4: 夹爪控制问题

#### 现象
```
阶段4: 闭合夹爪
self.set_gripper(0.0)  # 完全闭合
```

#### 根本原因
1. **夹爪闭合速度过快**: 瞬间闭合可能导致物体滑落
2. **没有力控制**: 只是位置控制，没有考虑抓取力
3. **没有接触检测**: 无法确认夹爪是否真正接触到物体

#### 修复方案

**方案1: 渐进式夹爪闭合**
```python
def close_gripper_gradual(self, steps: int = 10):
    """渐进式闭合夹爪"""
    initial_width = 0.03
    final_width = 0.0
    
    for i in range(steps):
        width = initial_width - (initial_width - final_width) * (i + 1) / steps
        self.set_gripper(width)
        self.step(50)  # 每步等待50步
```

**方案2: 添加力矩反馈**
```python
def get_gripper_torque(self) -> float:
    """读取夹爪力矩"""
    # 读取夹爪电机的力矩反馈
    # 这需要修改模型添加力矩传感器
    return self.data.actuator_force[6]  # 或其他传感器

def close_with_force_limit(self, max_torque: float = 0.5):
    """限制力矩的闭合"""
    while True:
        self.set_gripper(self.data.ctrl[6] - 0.001)  # 每次闭合0.001
        self.step(10)
        torque = self.get_gripper_torque()
        
        if torque >= max_torque:
            print(f"  达到最大力矩 {torque}，停止闭合")
            break
        
        if self.data.ctrl[6] <= 0.0:
            print("  夹爪已完全闭合")
            break
```
